### host端：

1. 添加网桥   
```
# brctl addbr br0
```

2. 让eth0成为br0的一个端口   
```
# brctl addif br0 eth0
```

3. 删除虚拟机ip地址    
```
# ip addr del dev eth0 192.168.100.37/24
```

4. 开启网桥     
```
# ifconfig br0 192.168.100.37/24 up
```

5. 添加路由   
```
route add default gw 192.168.100.1
```
